# Motivational Quote Generator (Flask)

A simple Flask web application that displays a random motivational quote every time the page is refreshed.

## Features
- Random quote generation
- Clean and minimal UI
- Flask backend
- Beginner-friendly project structure

## Technologies Used
- Python
- Flask
- HTML
- CSS
